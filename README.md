# amflow-util


このモジュールは、 Akashic Message Flow (AMFlow) のユーティリティモジュールです。
**ゲーム開発者(Akashic Engineの利用者)がこのモジュールを直接利用する必要はありません**。

## インストール

Node.jsが必要です。次のコマンドでインストールできます。

```
npm install @akashic/amflow-util
```

## ビルド方法

TypeScriptで書かれています。インストール後にビルドしてください。

```sh
npm install
npm run build
```

## 利用方法

`require()` してください。

```javascript
require("@akashic/amflow-util");
```

## テスト方法

```
npm test
```
